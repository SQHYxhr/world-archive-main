# 设定档案馆 / World Archive

一个本地优先的世界观与 OC 设定管理工具，用来整理角色、地点、组织、物品以及它们之间的关联。

## 在线体验

[https://lorefolio.vercel.app/](https://lorefolio.vercel.app/)

## 适合用来做什么

- 小说与故事的世界观设定整理
- OC 角色档案管理
- 地点、势力、道具的系统化记录
- 创作资料归档与查找
- 本地个人创作工作台

## 当前功能

### 世界项目

- 创建多个独立的世界 / 项目
- 每个世界内管理专属的设定条目

### 角色档案

- 外貌、性格、代称、身份、状态、代表台词
- 能力与技能、目标与执念
- 背景故事与补充设定
- 所属组织、常驻地点、种族的条目引用
- 角色间关系图谱

### 地点档案

- 地点分类、当前状态
- 环境氛围、标志性地标与建筑
- 历史沿革、出入方式与限制
- 上级地点层级、主要管辖组织

### 组织 / 势力档案

- 组织类型、当前状态
- 理念与目标、内部结构、影响力与势力范围
- 历史沿革、创作备注
- 上级组织、总部地点
- 自动派生显示成员角色、管辖地点、下属组织

### 物品 / 道具档案

- 物品类型、当前状态
- 外观描述、功能用途、材质构造
- 来源出处、历史沿革
- 使用限制与副作用、创作备注
- 当前持有者、当前所在地、制造 / 所属组织的条目引用

### 通用能力

- 富文本正文编辑（支持正文插图）
- 封面图与图集
- 标签系统
- 相关条目关联
- 收藏与置顶
- JSON 数据导出
- JSON 备份文件校验
- 备份文件导入恢复
- 导入前自动下载当前数据备份
- 首次访问提供示例世界（可自由修改或删除）

## 数据存储说明

本项目为本地优先应用，**设定数据保存在当前浏览器的 localStorage 中**，不会上传或同步到任何服务器。

以下操作可能导致看不到原来的数据：

- 清除浏览器数据（缓存 / Cookie / 站点数据）
- 更换浏览器或更换设备
- 使用无痕 / 隐私模式

建议定期使用页面上的「导出数据」功能保存 JSON 备份到本地文件。导入备份会覆盖当前浏览器中的数据，导入前系统会自动下载一份导入前数据作为备份。

## 本地运行

```bash
npm install
npm run dev
```

打开 http://localhost:3000

```bash
npm run build
npm start
```
